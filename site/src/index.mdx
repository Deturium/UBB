# YAUC

☀️ Yet Another UBB for CC98.

`YAUC` 是一个旨在提供一系列工具帮助开发者将 CC98 的 UBB Code 转换为 html 或者 ReactNode 等形式的项目。

其将作为 CC98 论坛的下一代 UBB 解析器。


## What's this ?

这里是 YAUC 的文档站点，在这里你可以：


#### 作为用户：

1. 学习 CC98 UBB Code 的写法

2. 查看每个标签的渲染结果

#### 作为开发者：

1. 了解 YAUC 如何解析 UBB Code

2. 了解如何使用 YAUC 来扩展你自己的 UBB 解析器


## How to use ?

这里由两部分组成：

`@cc98/ubb-core`

一个扩展性良好的 UBB 核心，提供解析 UBB 的核心能力。

具体见左侧 `UBB-CORE` 目录下内容。

<br />

`@cc98/ubb-react`

依托 core 基于 CC98 UBB 规范实现的 React 封装，开箱即用。

具体见左侧 `UBB-REACT` 目录下内容。
