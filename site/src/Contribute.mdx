# Contribute

我们欢迎任何人给我们提出任何建议


## Github

该项目的 Github 地址为： [YAUC](https://github.com/Deturium/UBB)

这是一个 monorepo，其中有三个 repo，他们都在 scope `@cc98` 下

- `@cc98/ubb-core`
- `@cc98/ubb-react`
- `@cc98/ubb-site`

其中 `@cc98/ubb-core` 和 `@cc98/ubb-react` 将发布在 `npm` 上，你可以轻松的获取到他们

`@cc09/ubb-site` 就是该文档站点，不在 `npm` 上发布
